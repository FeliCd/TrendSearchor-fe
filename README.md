# TrendSearchor — Frontend

> React 18 + Vite + TailwindCSS

## Yêu cầu

- [Node.js](https://nodejs.org/) (v18+)
- Backend đang chạy tại `http://localhost:8080`

## Setup & Chạy local

### 1. Cài dependencies

```bash
npm install
```

### 2. Tạo file `.env`

```bash
copy .env.example .env
```

> Mặc định đã trỏ đến `http://localhost:8080`, không cần sửa gì nếu backend chạy local.

### 3. Chạy dev server

```bash
npm run dev
```

Truy cập: **http://localhost:3000**
